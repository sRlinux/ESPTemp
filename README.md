# ESPTemp
DIY wireless LAN thermometer for homeautomation

## What you need:
### Hardware
* 1x ESP8266 ESP-01S
* 1x Dallas 18B20
* 1x Pullup Resistor (something like ~5k)
* A piece of stripboard
* Soldering iron
* Some solder
* optional: an UART USB Adapter @3,3V, ON/OFF switch

### Software
* Arduino IDE
* esptool
* AT firmware for ESP8266
* MQTT Broker (mosquitto)
